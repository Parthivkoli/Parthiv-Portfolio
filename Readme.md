# Parthiv Koli's Portfolio

Welcome to my personal portfolio! This website showcases my skills, projects, and professional experience as an aspiring Cloud and Data Analyst. Built with HTML, CSS, and JavaScript, it reflects my passion for technology and my journey in Computer Science and Engineering.

## About Me
I'm Parthiv Koli, a final-year B.Tech student in Computer Science and Engineering at Chhatrapati Shivaji Maharaj University, Navi Mumbai. I’m enthusiastic about cloud computing, data analytics, and artificial intelligence, with hands-on experience in Python, web development, and more. This portfolio is a window into my work and aspirations.

## Features
- **Responsive Design**: Adapts seamlessly to desktop and mobile devices.
- **Interactive Easter Egg**: A hidden "Anime Character Quiz" game triggered by clicking the profile image 10 times—test your anime knowledge!
- **Professional Sections**: Includes About Me, Experience, Education, Skills, Projects, and Contact.
- **Dynamic Content**: Fetches anime character images from the Jikan API for the quiz game.
- **Resume Link**: Access my detailed resume via a dedicated page.

## Technologies Used
- **HTML5**: Structure and content.
- **CSS3**: Styling, animations (via Animate.css), and responsive design.
- **JavaScript**: Interactivity, game logic, and API integration.
- **External Libraries**:
  - [Font Awesome](https://fontawesome.com/) for icons.
  - [Animate.css](https://animate.style/) for animations.
- **API**: [Jikan API](https://jikan.moe/) for fetching anime character data.

## Setup Instructions
To run this portfolio locally:
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/Parthivkoli/Parthiv-Portfolio.git
   cd Parthiv-Portfolio
   ```
2. **Open the Project**:
   - Open `index.html` in a web browser (e.g., Chrome, Firefox).
   - No additional setup is required as it’s a static site.

3. **Explore the Easter Egg**:
   - Click the profile image 10 times to launch the "Anime Character Quiz" game.
