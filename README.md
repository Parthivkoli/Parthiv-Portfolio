# Parthiv Koli's Portfolio

Welcome to my personal portfolio website, hosted on GitHub Pages! This portfolio showcases my projects, skills, and experiences in the field of Artificial Intelligence, Machine Learning, and Data Analytics. It's a place where I highlight my professional journey, completed internships, and key achievements.

## 🚀 Website Link

[Visit my portfolio here](https://parthivkoli.github.io/Parthiv-Portfolio/)

## 🛠️ Technologies Used

- **HTML5**: For the structure and layout of the website.
- **CSS3**: To style the webpage, making it visually appealing.
- **JavaScript**: For adding interactive elements to the website.
- **GitHub Pages**: To deploy and host the portfolio online.
  
## 💼 Sections

The portfolio is divided into the following sections:

- **Home**: A brief introduction about myself and my goals.
- **About Me**: Details about my background, education, and professional experience.
- **Projects**: Highlights of key projects I’ve worked on, including AI, ML, and data analytics projects.
- **Internships**: Information about the internships I’ve completed, including at CodSoft, Prodigy InfoTech, and Connecting Dreams Foundation.
- **Contact**: A simple form and links to reach out to me.

## 🔥 Key Features

- **Responsive Design**: The website is fully responsive, ensuring a seamless experience across devices, including mobile, tablet, and desktop.
- **Smooth Navigation**: Easy-to-use navigation bar to quickly access different sections of the portfolio.
- **Project Gallery**: Each project is presented with a brief description, tools/technologies used, and a link to the project (if applicable).
